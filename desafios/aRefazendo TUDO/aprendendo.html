Lista de definições

A lista de definições vem da tag <dl>, que significa Definition List {lista de definição}.

A tag <dt>,significa "Termo", ela fica um pouco mais na esquerda, já a TAG <dd>, fica um pouco mais para a direita.

A vantagem dessa Tag é que ela ajuda nos mecanismo de busca do Google, e ele dá valor para sites que usa Lista de definições. Como por exemplo quando alguém pesquisa "Qual a definição de HTML?", o Google vai buscar site que tem Lista de Definições.

Exemplos:
HTML
Linguagem de marcação para criação de site do conteúdo de um site.
CSS
Linguagem de marcação para a criação do desing de um site.
JavaScript
Linguagem de programação para a criação da interatividade de um site.
